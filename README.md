# 灰狼优化算法(GWO)路径规划

# Grey Wolf Optimizer

### 用法：

- matlab运行main.m
- 自带三种UAV_SetUp.m仿真参数设置（也可自行设计仿真环境）
- 同时支持2D无人机轨迹规划和3D无人机轨迹规划（取决于UAV_SetUp.m的参数设置）
- 自带3种启发搜索算法（在main.m中选择）
- 可在ObjFunc.m中调整目标函数权重等
- 程序运行较慢，但有进度条显示，耐心等待
